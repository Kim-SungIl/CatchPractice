# CatchPractice
Catch 멘토링 스프링 서버개발 실습 1주차

## Java 설치 확인
 - 윈도우
   - 실행 > cmd > java -version
 - Mac OS
   - 터미널 > java -version
 - java version 1.8이상이 설치되어 있을 경우 다음으로 진행
 - 설치가 되어있지 않을 경우, https://java.com/ko/download/ 설치 진행
 
## Eclipse download
